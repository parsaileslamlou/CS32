# UCLA CS 32 Projects (Fall 2025)

This repository contains my implementations of  C++ projects for UCLA's CS 32 course, demonstrating  skills in **object-oriented programming, data structures, dynamic memory management, and C++ design**.  

---

## Projects Overview

### **Project 1: Blood Donation & Vacation Credit Management System**
- **Summary:** HR system to track employee blood donations and automatically credit vacation hours.  
- **Key Skills:** C++ classes, **dynamic memory with raw pointers**, Rule of Three, modular design, date validation, deterministic return codes.  
- **Highlights:** Accurate business logic enforcement, edge-case handling (leap years, exact spacing), using tests validating functionality.  

---

### **Project 2: TaskList with Doubly Linked List**
- **Summary:** Custom doubly linked list implementation for a to-do list ADT.  
- **Key Skills:** **Pointers and dynamic memory**, Rule of Three (deep copy, assignment, destructor), linked list operations (add, remove, find, get), memory-safe design.  
- **Highlights:** Full coverage of edge cases, robust memory management, no STL containers used, clean modular design.
